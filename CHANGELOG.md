# Changelog

## v1.0.0 - Initial Release (2026-04-13)

Unstable Rifts launches as a full top-down co-op roguelike shooter experience for Hytale.

### Highlights

* Added the full dungeon run gameplay loop for 1-4 players, built around generated routes, combat rooms, side paths, loot rooms, shops, and a final boss encounter.
* Added party-based co-op with public and private parties, invites, a dedicated party UI, shared run coins, and reconnect support during active dungeon runs.
* Added 19 weapons with distinct roles and feel, including ranged, melee, utility, and elemental options.
* Added 6 armor sets across 24 pieces, with set bonuses and stat-driven build variety.
* Added loot progression through crates, rarity tiers, modifiers, and equipment upgrades earned during runs.
* Added top-down camera gameplay, in-run HUD systems, revive flow, shop interactions, and portal-based dungeon access.

### Gameplay

* Fight through Industrial enemies, infected DeadWood, wildlife encounters, and a full boss fight at the end of the current dungeon level.
* Explore replayable dungeon layouts with main paths and optional branches instead of a single fixed encounter sequence.
* Break crates to find weapons, armor, healing, ammo, coins, and other run-saving upgrades.
* Spend shared team coins in shop rooms to improve the party mid-run.
* Revive teammates during combat and keep pushing deeper with your group.

### Multiplayer and Systems

* Added support for solo play and co-op parties of up to 4 players.
* Added party creation, invites, joining, privacy settings, kicking, leaving, disbanding, and public party listing.
* Added reconnect handling so disconnected players can rejoin an active run without losing their party slot.
* Added HUD and interface support for party play, run state awareness, and core dungeon interactions.

### Commands and Admin Tools

* Added player commands for party management, dungeon entry, HUD toggling, and top camera control.
* Added admin tools for portal spawning, random loot testing, team coin management, and pickup debugging.

### Content Overview

* 19 weapons
* 6 armor sets (24 pieces)
* 1 dungeon level
* 1 boss
* 14+ enemy types
